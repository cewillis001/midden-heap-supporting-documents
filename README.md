# Midden Heap Supporting Documents

Supporting documents for Midden Heap posts.

Documents are stored as Markdown and organized by post title under `posts/`.

## Structure

```text
posts/
  bar-exam-benchmark-followup-essay/
    README.md
    tables/
```

Use one folder per post. Keep converted tables, source notes, and other supporting materials inside that post's folder.
